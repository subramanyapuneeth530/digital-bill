# Digital Menu & Billing System

A modern, mobile-first digital menu and billing system for hotels and restaurants. Customers can scan a QR code to view the menu, place orders, and track their bill in real-time.

## 🚀 Features

- **Digital Menu**: Interactive and visually appealing menu with category filtering.
- **Real-time Billing**: Customers can see their running total as they add items.
- **Service Requests**: Call a waiter or request the bill directly from the interface.
- **Responsive Design**: Optimized for all devices, especially mobile smartphones.
- **Admin Dashboard**: (In Progress) View and manage incoming orders.

## 🛠️ Tech Stack

- **Frontend**: React.js with Vite
- **Styling**: Vanilla CSS (Modern, Responsive)
- **State Management**: React Hooks

## 📥 Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## 🏗️ Project Structure

- `src/pages/Menu.jsx`: The customer-facing menu interface.
- `src/pages/Dashboard.jsx`: The restaurant management dashboard.
- `src/App.jsx`: Main application routing and logic.
- `src/index.css`: Global styles and design system.
